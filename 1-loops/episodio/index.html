Loops em Javascript

Map, Filter e Reduce

Transformando itens dos arrays

  elementos.map(function(elemento) {
    ...
    return [ELEMENTO_NOVO];
  })

Filtrando itens dos arrays

  elementos.filter(function(elemento) {
    ...
    return [BOLEANO];
  })

Transformando os arrays

  elementos.reduce(function(acumulador, elemento) {
    ...
    return [ELEMENTO_NOVO];
  }, [INICIALIZADOR])

Faixas

['branca', 'azul', 'roxa', 'marron', 'preta', 'vermelha-preta', 'vermelha-branca', 'vermelha']
